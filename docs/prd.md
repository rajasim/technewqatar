# Techneqatar Website Requirements Document

## 1. Website Overview

### 1.1 Website Name
Techneqatar Official Website

### 1.2 Website Description
A professional corporate website for Techneqatar, a construction company specializing in government and infrastructure projects in Qatar. The website serves as a digital showcase for the company's capabilities, certifications, and project portfolio, targeting Qatar Government, Semi-Government Authorities, and Large Corporates.

### 1.3 Target Audience
- Qatar Government entities
- Semi-Government Authorities
- Large Corporate clients
- Potential business partners
- Job seekers in construction industry

## 3. Design Style

### 3.1 Color Scheme
- Primary colors: Dark Blue and Maroon\n- Background: White\n- Accent colors: Gold for highlights (Qatar-friendly palette)
\n### 3.2 Visual Elements
- Clean, modern corporate layout
- Professional sans-serif typography (e.g., Inter, Roboto, or Open Sans)
- Minimal animations, focus on content clarity
- High-quality project imagery (no stock-looking visuals)
- Subtle shadows for depth and card-based layouts
- Consistent spacing and grid system for visual balance

### 3.3 Layout Style
- Card-based layout for projects and services

- Full-width hero sections with overlay text
- Structured content hierarchy suitable for government tender presentations
